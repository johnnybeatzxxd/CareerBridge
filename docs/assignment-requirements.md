# Final Project Assignment Requirements

Source: `Final Project Assignmmnet.docx`

## Project

Developing Interactive Job Site

## Users

- Job seeker
- Employer/company
- Admin

## Functional Requirements

### Job Seeker

- Login/logout and register
- Manage account (CRUD)
- View dashboard
- Submit application
- View company email on job post
- Upload CV (optional)
- Build CV (CRUD)
- Download CV
- View jobs
- Search jobs
- Filter job search
- View landing page
- Get job alert (optional)
- View applied jobs

### Employer

- Login/logout and register
- Manage account (CRUD)
- Post a job
- Manage job
- View dashboard
- View applications

### Admin

- Manage system
- Prepare CV template
- Manage users (CRUD)
- Approve employer
